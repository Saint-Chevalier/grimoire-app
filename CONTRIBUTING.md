# Contributing to Grimoire (Constellation Codex)

Thank you for caring about this tool. We recruit for **action under pressure**, not résumés.

## What this project is

A **local-first spellcraft surface** for talking *about* one focus (AI node or person), densening their intelligence map, and crafting messages (spells) you send yourself.

You are the transport layer. The app is the codex.

## What we want help with

- UI clarity without killing the myth layer (constellation / stages)
- Spell quality (copy people would actually send)
- Accessibility + keyboard flows
- Tests / deterministic fixtures for spell + alignment parsers
- Packaging / install / GitHub Pages demos
- Documentation that teaches the **method**, not private kingdoms

## What we reject

- Shipping private vault dumps, personal dossiers, secrets, or biometrics
- Features that merge two focuses into one channel
- “Auto-send to the internet” without explicit user copy/mark-sent control
- CSS fireworks that block the chat input

## Ladder (human circle)

| Stage | Meaning |
|-------|---------|
| **User** | Runs the app, opens issues |
| **Contributor** | Ships PRs that land green |
| **Circle candidate** | Pattern of truth + reliable execution |
| **Knight / squad** | Trusted operator in the Discord Empire |

Standards live with the operator and the Trust Ledger. Code reviews stay technical.

## How to contribute

1. Fork / branch from `main`
2. Keep PRs small and description-first: **problem → change → how to test**
3. Hard-refresh after JS/CSS edits when using a local static server
4. Do not commit `GRIMOIRE-FocusIntelligence/` or machine-local paths

## Local run

```bash
# from repo root
npx --yes serve .
# or
python -m http.server 5173
```

Open the URL shown. ES modules require HTTP (not `file://`).

## Code of conduct

Be precise. Be kind under pressure. Correct wrong claims without theater. Disagreement is signal; sabotage is expulsion.

## Security

If you find a vulnerability, open a private report or contact the maintainers via the Discord path listed in the README. Do not dump exploit PoCs publicly without coordination.
