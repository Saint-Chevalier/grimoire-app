# Roadmap

Public method roadmap for **Grimoire — Constellation Codex**.  
Private operator doctrine is never required to run or contribute.

## Shipped (v0 public brick)

- Sealed Focus model (1 Focus = 1 receiving entity)
- Local chat + paste intelligence
- Spell panel: forge / copy / mark sent
- Universe densening + optional Intel Atlas inspect
- MIT license, contributor ladder, local-first vault option

## Next — Chrono-Ring (read-only)

**Stars show where memory densens. Chrono-Ring shows when truth became true.**

Planned as a **read-only** ordered view of Focus events (messages, spells, milestones) built only from existing local timestamps.

### Guarantees
- No automatic outbound messaging
- No silent disk mutation from the ring
- No requirement to upload private life data to a cloud

### Out of scope for first Chrono-Ring drop
- Clinical relationship diagnosis
- Mandatory “love language” profiling
- Cloud multiplayer memory

## Later candidates

- Better public starter spellbook quality
- GitHub Pages / static demo host
- Optional person-Focus soft tags (private-by-default hypotheses)
- Contributor UX polish
